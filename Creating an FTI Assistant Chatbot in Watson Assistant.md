Next are the steps in the process of creating an FTI Assistant Chatbot using the Watson Assistant platform. This 
chatbotis designed to provide an interactive and efficient solution for users in accessing information and services 
available at the Faculty of Information Technology (FTI).

# Watson Assistant
  1. Select Watson Assistant on the resource list menu, then Launch Watson Assistant
  2. Next, input the Assistant Name "FTI Assistant", for Description "Optional", and for Language select "English (US)",
     then Next
  3. Then set the Watson Assistant deployment according to your needs, click next, then Create
  4. The following is a display of the project in Watson Assistant that was successfully created
  5. Next, integrate Watson Assistant by clicking the "Integrations" menu
  6. Click Build Custom Extension and click Next
  7. Enter the extension name and description "Optional" then Next
  8. Download the Json Extensions at the following link: [Json](https://github.com/watsondeveloper-cloud/assistant-toolkit/blob/master/integrations/extensions/starterkits/language-model-watsonx/watsonx-openapi.json)
  9. Enter the downloaded json file then click Next, and Finish
  10. After creating Extensions, the next step is to "Add" the Watson Extensions. Click Confirm, Next 2 times, and
      Finish as shown in the image below
